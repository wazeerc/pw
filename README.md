# pw

## **Getting Started**

### **Prerequisites**

- [Go](https://go.dev/dl/) installed on your system.

### **Installation**

1. Clone the repository

2. Run the program:
   ```
   cd pw && go run main.go gen -l 16 -n
   ```
   or
   ```
    go build && go install
    pw gen -l 16 -n
   ```

3. To use the interactive mode:
   ```
   pw -i
   ```

### **Help**

To get help on how to use the program, run:
```
pw help
```
